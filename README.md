## рҹҢҹ About the Project
DermaBridgeX allows users to enter basic details like age, gender, 
skin type, and concerns to receive suitable product suggestions with 
direct purchase links. A smart appointment system supported by a 
chatbot ensures smooth scheduling. Secure online payments and 
real-time video consultations make the platform reliable and 
convenient for users.
